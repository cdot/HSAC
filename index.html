Filter lifetime predictions are made based on the temperature of the intake air at runtime. The predicted lifetime is the lifetime of a filter that is run at the optimum temperature (usually 20°C). The coefficients are for the filter temperature degradation curve. The sensor IDs are the unique identifiers for the different environmental sensors. See https://github.com/cdot/HSAC for more.
Two databases are used:
  • A read-only online database, accessed through the URL of a CSV file that contains the root index
  • A read-write cache server
Click "Update" to update the cache server from the online database. See https://github.com/cdot/HSAC for more
